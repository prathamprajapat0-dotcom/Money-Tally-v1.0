Money Tally App 💰
Description

Money Tally is a lightweight desktop application built using Python and Tkinter. It allows users to enter and track values in real time with a simple and easy-to-use interface.

Features

Real-time input (no button required)

Multiple entry support

Clean and user-friendly UI

Fast and lightweight performance

Works without installing Python

How to Use

Download the .exe file

Double-click to open the application

Start entering values directly

Requirements

Windows PC (no additional software needed)

Notes

This app is fully standalone and portable. You can run it on any compatible system without setup.
